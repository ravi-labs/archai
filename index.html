ArchAI generates diagram source from a description and renders it. For drawio, it loads into the embedded drawio editor. For Mermaid, it renders client-side via the Mermaid library.

You can also load an existing diagram source file (drawio .drawio/.xml or Mermaid .mmd) and ask the AI to refine it with natural-language instructions.

Built on the open-source drawio editor and Mermaid. ArchAI is not affiliated with JGraph Ltd or the Mermaid project.
